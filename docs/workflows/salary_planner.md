# salary_planner

A workflow prompt for HRA exemption, income tax comparison, and take-home salary planning.

**Use case:** combine `calculate_hra_exemption` with `calculate_income_tax` to estimate monthly take-home pay.

**Example prompt:** "Plan salary for 24 lakh annual CTC with HRA and tax comparison"

**Notes:** This is a prompt workflow, not a callable tool. It is meant to guide tax planning conversations.
